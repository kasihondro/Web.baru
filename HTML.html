Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web. HTML memberikan kerangka dasar bagi konten di internet dan memungkinkan pengembang web untuk menampilkan teks, gambar, video, dan elemen multimedia lainnya.

Fungsi Utama HTML

  • Menentukan struktur halaman web.
  • Mengatur konten dengan elemen seperti heading, paragraf, dan daftar.
  • Menyisipkan gambar dan video.
  • Membuat tautan (link) antar halaman.
  • Mendukung formulir untuk interaksi pengguna.

Struktur Dasar HTML

Berikut adalah struktur dasar dari dokumen HTML:

<!DOCTYPE html>
<html lang="id">
    <head>
        <meta charset="UTF-8">
        <title>Judul Halaman</title>
    </head>
    <body>
        <h1>Selamat Datang di Halaman Web</h1>
        <p>Ini adalah paragraf pertama di halaman ini.</p>
    </body>
</html>

Kesimpulan

HTML adalah fondasi dari setiap halaman web. Memahami HTML adalah langkah pertama yang penting bagi siapa saja yang ingin belajar tentang pengembangan web.
