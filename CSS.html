Pengenalan CSS

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mendesain dan mengatur tampilan halaman web. CSS memungkinkan pengembang untuk mengontrol layout, warna, font, dan elemen visual lainnya dari halaman web secara terpisah dari struktur HTML.

Fungsi Utama CSS

  • Mengatur tata letak dan desain halaman web.
  • Menentukan warna, font, dan ukuran teks.
  • Menerapkan gaya pada elemen HTML secara global atau spesifik.
  • Membuat responsif desain untuk berbagai perangkat.
  • Menggunakan efek transisi dan animasi untuk meningkatkan interaksi pengguna.

Contoh CSS

Berikut adalah contoh kode CSS sederhana:

/* Mengatur gaya untuk seluruh halaman */
body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}

/* Mengatur gaya untuk heading */
h1 {
    color: #333;
    text-align: center;
}

/* Mengatur gaya untuk paragraf */
p {
    color: #555;
    line-height: 1.5;
}

Kesimpulan

CSS adalah alat yang sangat penting dalam pengembangan web modern. Dengan menggunakan CSS, pengembang dapat menciptakan tampilan yang menarik dan responsif, meningkatkan pengalaman pengguna di situs web.
