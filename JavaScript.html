Pengenalan JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif. JavaScript berjalan di sisi klien (browser) dan memungkinkan pengembang untuk menambahkan berbagai fitur dinamis, seperti animasi, validasi formulir, dan interaksi pengguna tanpa perlu memuat ulang halaman.

Fungsi Utama JavaScript

  • Membuat konten interaktif dan dinamis di halaman web.
  • Menangani event (peristiwa) seperti klik, hover, dan input pengguna.
  • Memanipulasi elemen HTML dan CSS secara langsung.
  • Berkomunikasi dengan server menggunakan AJAX untuk memuat data tanpa memuat ulang halaman.
  • Membuat aplikasi web yang kompleks dengan menggunakan framework seperti React, Angular, atau Vue.js.

Contoh Kode JavaScript

Berikut adalah contoh kode JavaScript sederhana yang menampilkan pesan selamat datang ketika tombol diklik:

<!DOCTYPE html>
<html lang="id">
<head>
    <title>Contoh JavaScript</title>
    <script>
        function tampilkanPesan() {
            alert("Selamat datang di halaman web kami!");
        }
    </script>
</head>
<body>
    <button onclick="tampilkanPesan()">Klik Saya</button>
</body>
</html>

Kesimpulan

JavaScript adalah bahasa pemrograman yang sangat penting dalam pengembangan web modern. Dengan kemampuannya untuk membuat halaman web interaktif dan responsif, JavaScript menjadi salah satu alat utama bagi pengembang web.
